# AI-First CRM HCP Module

This project is a prototype AI-powered CRM system for Healthcare Professionals (HCP).

## Tech Stack
- Frontend: React + Redux
- Backend: FastAPI
- AI: LangGraph + LLM (Groq)

## Features
- Chat-based interaction logging
- Form-based logging
- Edit & fetch interactions
- AI-powered summarization

## Note
This is a prototype focusing on architecture and AI integration.
